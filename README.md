# SmallStep Foundation

The SmallStep Foundation empowers the GNUStep ecosystem by creating small, focused, and thoughtfully crafted applications that fill essential everyday gaps. We champion simplicity, openness, and usability — delivering tools that are lightweight, consistent, and true to the spirit of the GNUstep and NeXTSTEP traditions. By taking many small steps, we help the community move forward together.
